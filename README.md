# JS Shopping Cart Tutorial

 Learn vanilla JavaScript by creating a shopping system. This tutorial covers many of the core concepts of vanilla JS along with a couple advanced topics. 

 - Variables 
 - Functions 
 - Arrays
 - Objects 
 - Events
 - and more!
